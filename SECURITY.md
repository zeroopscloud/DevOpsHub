# Security Policy

## Supported Versions

The latest version of DevOpsHub is actively supported.
Older versions may not receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT create a public issue.

Instead, report it by:
- Contacting the maintainers directly
- Or emailing: zeroopscloud@gmail.com (replace with real email)

We will review the report and respond as soon as possible.
